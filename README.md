зј–иҫ‘ `.env` й…ҚзҪ®ж–Үд»¶пјҡ

```env
API_ELF_URL="https://github.com/FloatCTF/floatctf/releases/latest/download/floatctf-linux-amd64-musl"

# API CONF
API_SERVER_IP="floatctf-api"
API_SERVER_PORT=9090
API_USER="floatctf_api"

# nginx
NGINX_SERVER_HTTP_PORT=80
NGINX_SERVER_HTTPS_PORT=443

# database
PG_HOST="floatctf-db"
PG_PORT=5432

# rustfs
RUSTFS_ACCESS_KEY=rustfsadmin
RUSTFS_SECRET_ACCESS_KEY=rustfsadmin

DOCKER_HOST_PATH="/var/run/docker.sock"
```

## ж ёеҝғеҠҹиғҪ

### з”ЁжҲ·з«Ҝ

- **зҷ»еҪ•жіЁеҶҢ** вҖ” еӯҰеҸ·жіЁеҶҢгҖҒJWT д»ӨзүҢйүҙжқғгҖҒArgon2 еҜҶз ҒеҠ еҜҶгҖҒеҜҶз ҒйҮҚзҪ®
- **йҰ–йЎө / еӨ©жўҜжҺ’иЎҢжҰң** вҖ” е®һж—¶еұ•зӨәи§ЈйўҳжҺ’еҗҚпјҢй•ҝжңҹз§ҜзҙҜжҝҖеҸ‘иҮӘдё»еӯҰд№ еҠЁеҠӣ
- **йўҳеҚ• / еҒҡйўҳ** вҖ” жҢү Web / Pwn / Crypto / Reverse / Misc еҲҶзұ»жөҸи§ҲпјҢзӮ№еҮ»ејҖеҗҜеҚіиҮӘеҠЁеҲӣе»әзӢ¬з«Ӣ Docker е®№еҷЁпјӣж”ҜжҢҒж•ҷеёҲеҸ‘еёғ"йўҳеҚ•"з»„еҗҲдё“йЎ№и®ӯз»ғ
- **Discussion и®Ёи®ә** вҖ” еңЁзәҝи®әеқӣпјҢж”ҜжҢҒеҗҢеӯҰй—ҙиҝӣиЎҢеҪјжӯӨеӯҰд№ дәӨжөҒ
- **жҜ”иөӣ / з§ҜеҲҶзңӢжқҝ** вҖ” ж”ҜжҢҒ JeopardyпјҲи§Јйўҳиөӣпјүе’Ң AWDпјҲж”»йҳІеҜ№жҠ—пјүдёӨз§ҚиөӣеҲ¶пјҢжҸҗдҫӣе®һж—¶з§ҜеҲҶзңӢжқҝгҖҒеҫ—еҲҶи¶ӢеҠҝеӣҫгҖҒдёҖиЎҖж Үи®°гҖҒиөӣдәӢе…¬е‘Ҡ

### з®ЎзҗҶз«Ҝ

- **иөӣдәӢжҰӮи§Ҳ** вҖ” еҸҜи§ҶеҢ–зі»з»ҹзҠ¶жҖҒзңӢжқҝпјҢе®һж—¶еұ•зӨәжңҚеҠЎеҷЁиҙҹиҪҪгҖҒеҶ…еӯҳ/зЈҒзӣҳдҪҝз”ЁзҺҮгҖҒзҪ‘з»ңжөҒйҮҸ
- **иөӣдәӢз»ҶиҠӮз®ЎзҗҶ** вҖ” йўҳзӣ®еўһеҲ ж”№жҹҘгҖҒDocker й•ңеғҸй…ҚзҪ®гҖҒз«ҜеҸЈжҳ е°„гҖҒйҷ„д»¶дёҠдј гҖҒз§ҜеҲҶи§„еҲҷй…ҚзҪ®
- **ж—Ҙеҝ—** вҖ” ж“ҚдҪңж—Ҙеҝ—дёҺе®Ўи®Ўи®°еҪ•жҹҘиҜў
- **Docker** вҖ” жҹҘзңӢжүҖжңүиҝҗиЎҢдёӯзҡ„е®№еҷЁе®һдҫӢпјҢж”ҜжҢҒејәеҲ¶й”ҖжҜҒејӮеёёе®№еҷЁпјҢйҮҠж”ҫжңҚеҠЎеҷЁиө„жәҗ
- **Tasks** вҖ” д»»еҠЎйҳҹеҲ—з®ЎзҗҶдёҺи°ғеәҰ

## еёёз”Ёе‘Ҫд»Ө

```bash
# жҹҘзңӢж—Ҙеҝ—
docker compose --env-file ./.env --env-file ./app/.env logs -f

# жҹҘзңӢжҢҮе®ҡжңҚеҠЎж—Ҙеҝ—
docker compose --env-file ./.env --env-file ./app/.env logs -f floatctf-api

# йҮҚеҗҜжңҚеҠЎ
docker compose --env-file ./.env --env-file ./app/.env restart

# еҒңжӯўжңҚеҠЎ

# йҮҚе»әе№¶йҮҚеҗҜ

```
